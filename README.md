## Running locally

Clone the project

```bash
  git clone https://github.com/felipefreitasa/water-monitoring.git
```

Enter the project directory

```bash
  cd water-monitoring
```

Install the dependencies

```bash
  yarn
```

Start the server

```bash
  npx expo start
```
